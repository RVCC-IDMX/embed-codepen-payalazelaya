Pen: Pure CSS Gradient Background Animation

by Paola Ayala Zelaya

In this pen, the author was able to use HTML and CSS in order to create an animation where the background can change gradient colors. Each 15 seconds, the background gradient color switches color from blue to light purple to a slightly dark pink to a light pink and vice versa.
